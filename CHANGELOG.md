# Changelog

本文件记录《雾铃小镇》仓库中每一个对外可追踪版本的主要变化。

记录方式参考 Keep a Changelog，版本号采用 Semantic Versioning：

- `MAJOR`：结构性重做或不兼容的内容调整
- `MINOR`：新增一段完整可试玩内容、明显扩展体验范围
- `PATCH`：修复、润色、布局优化、音频与文档类更新

## [Unreleased]

### Added

- 第五天起开放“桥边共享风痕”面板与留痕弹层，允许玩家把一句短话挂到桥边。
- 新增 `shared-world.js` 与 `shared-world-config.js`，为 Supabase 异步共享桥边痕迹提供可选接入层，并保留本机 fallback。
- 新增桥边共享纸签 / 布条的像素环境演出，以及阿栀、林麦、沈砚对“外乡人痕迹”的轻量环境回应。
- 新增异步共享世界规划文档与 Supabase SQL / RLS 模板。

## [0.3.0] - 2026-04-22

### Added

- 七天完整试玩主线：从初到小镇一路推进到风铃集会当天。
- 5 位核心 / 氛围村民互动，以及村口、广场、面包房、桥边等关键片区对白。
- 公告栏、旧风铃试听、细雨找种、绑布条、试灯、风记签等完整任务链。
- 自动存档、章节导览、试玩总结、重复游玩与底部玩法指引。
- 主背景音乐、环境音、脚步声、对话声、任务提示音与桥边关键铃声。
- 对外试玩封面页、移动端触屏操作与 GitHub Pages 自动部署。
- 基于 `VERSION`、`CHANGELOG.md`、标签工作流的规范发布基础设施。

### Changed

- 桌面端与移动端的游戏区布局分开优化，移动端更强调操作顺序，桌面端更强调整体排版。
- 主 BGM 切换为 `assets/audio/hjm-hjx.mp3`，并调低音量，让环境音和提示音更清楚。
- 发布页文案与试玩入口整理为更适合 GitHub Pages / 对外试玩展示的一版。
