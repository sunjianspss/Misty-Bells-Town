# 雾铃小镇

这是一个零依赖的网页试玩 Demo，用来体验《雾铃小镇》第一周 Vertical Slice 的完整节奏：

- 村口遇见阿栀
- 面包房接到“送第一篮点心”的任务
- 广场交付点心
- 傍晚去桥边和沈砚听第一声旧铃
- 第二天查看公告栏，正式得知春季风铃集会
- 第二次送点心，把节日气氛往前推
- 第三天遇见“旧风铃只响半声”，并替许槐试听左、中、右三处风铃声
- 第四天在细雨里帮秦婆婆找回“雨后铃兰”种子
- 第五天帮阿栀把三条春色布条系到桥边，让节日前的忙碌重新抬起来
- 第六天在节日前夜替桥边试亮三盏灯，让“真的快到了”的感觉落下来
- 第七天在风铃集会当天收三句风记签，把这一周等来的第一声真正挂给人听
- 以“今日小记”和试玩总结收束前七天 Demo

- 现在项目已经接入一层“异步共享痕迹”骨架：从春 5 开始，玩家可以在桥边留下短句风痕
- 默认状态下先以本机模式运行，方便直接试玩与验证 UI
- 一旦把根目录 [`shared-world-config.js`](shared-world-config.js) 填上 Supabase 配置并开启 `enableRemote`，桥边风痕就会切到真正的公共共享模式
- 落地方案见 [`docs/async-shared-world-plan.md`](docs/async-shared-world-plan.md)
- Supabase 表结构与 RLS 模板见 [`docs/supabase-bridge-traces.sql`](docs/supabase-bridge-traces.sql)

## 七天记录

1. 第一天：初到雾铃小镇，先认识阿栀，再帮林麦把第一篮点心送去广场，傍晚第一次去桥边听风。
2. 第二天：公告栏正式贴出“春季风铃集会”，再送第二篮点心，把节日气氛往前推一步。
3. 第三天：旧风铃只响半声，替许槐试听左、中、右三处声音，让桥边那一句重新完整起来。
4. 第四天：细雨压低了小镇的声响，帮秦婆婆找回“雨后铃兰”种子，让节奏从准备转进人物走心。
5. 第五天：雨后初晴，帮阿栀把三条春色布条系到桥边，让安静之后的忙碌重新抬起来。
6. 第六天：节日前夜，替沈砚试亮三盏桥灯，把“开始忙起来了”推进成“真的快到了”。
7. 第七天：风铃集会当天，替阿栀收三句想让风记住的话，写成风记签挂上桥边，最后听见这一周真正等来的第一声。

## 直接打开

可以直接用浏览器打开 `index.html`。

如果你更习惯本地静态服务，也可以在项目目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://127.0.0.1:8000`。

## 操作

- `WASD` / 方向键：移动
- `E` / 空格 / 回车：交谈与互动
- 手机上可以使用页面下方的触屏按钮

## 配置 Supabase 共享风痕

1. 在 Supabase SQL Editor 里执行 [`docs/supabase-bridge-traces.sql`](docs/supabase-bridge-traces.sql)
2. 打开根目录 [`shared-world-config.js`](shared-world-config.js)
3. 填入：
   - `supabaseUrl`
   - `supabaseAnonKey`
   - 把 `enableRemote` 改成 `true`
4. 推送到 `main` 后，GitHub Pages 上的桥边共享风痕就会切到公共模式
